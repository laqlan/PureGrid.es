# PureGrid - Public Assets

This folder contains all the public assets for the PureGrid website.

## Logo

Place your logo as:
- `logo.png` - Main logo (recommended size: 200x60px or similar aspect ratio)

The logo will be displayed in:
- Header/navigation bar
- Footer

## Service Images

Create a `services` folder and add the following images:
- `services/digital-strategy.png`
- `services/brand-identity.png`
- `services/web-development.png`
- `services/ui-ux-design.png`
- `services/seo-marketing.png`
- `services/content-creation.png`

Recommended size: 800x600px or 16:9 aspect ratio

## Portfolio Images

Create a `portfolio` folder and add the following images:
- `portfolio/ecommerce.png`
- `portfolio/brand-redesign.png`
- `portfolio/mobile-app.png`
- `portfolio/marketing-campaign.png`
- `portfolio/saas-dashboard.png`
- `portfolio/content-strategy.png`

Recommended size: 800x600px or similar

## Notes

- All images are optional. The site will display beautiful gradient fallbacks if images are not present.
- PNG or JPG formats are recommended.
- Optimize images for web to ensure fast loading times.
- You can swap any image at any time by simply replacing the file with the same name.
