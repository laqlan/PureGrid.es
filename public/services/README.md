# Service Images

Place your service images in this folder with the following names:

1. `digital-strategy.png` - For Digital Strategy service
2. `brand-identity.png` - For Brand Identity service
3. `web-development.png` - For Web Development service
4. `ui-ux-design.png` - For UI/UX Design service
5. `seo-marketing.png` - For SEO & Marketing service
6. `content-creation.png` - For Content Creation service

**Recommended specifications:**
- Format: PNG or JPG
- Size: 800x600px (or any 4:3 / 16:9 aspect ratio)
- Optimized for web

The website will work perfectly even without these images, showing elegant gradient backgrounds as fallbacks.
