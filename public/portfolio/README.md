# Portfolio Images

Place your portfolio project images in this folder with the following names:

1. `ecommerce.png` - E-Commerce Platform project
2. `brand-redesign.png` - Brand Redesign project
3. `mobile-app.png` - Mobile App project
4. `marketing-campaign.png` - Marketing Campaign project
5. `saas-dashboard.png` - SaaS Dashboard project
6. `content-strategy.png` - Content Strategy project

**Recommended specifications:**
- Format: PNG or JPG
- Size: 800x600px (or any 4:3 / 16:9 aspect ratio)
- Optimized for web

The website will work perfectly even without these images, showing elegant gradient backgrounds as fallbacks.
